# Security Policy

Please report security issues privately to **privacy@sliceops.org**. Do not open a
public issue for a vulnerability.

Include, where you can: a description, the affected file or template, reproduction
steps, and the impact you observed.

## Scope

This repository is the SliceOps toolkit: stdlib-only Python (consistency validators,
the band-calibration script) and CI-guardrail templates. In scope are the scripts and
templates here. The framework specification lives in
[SliceOps/spec](https://github.com/SliceOps/spec).

## What to expect

- Acknowledgement within a few business days.
- An honest severity assessment and a fix timeline, in coordinated disclosure with you.
- Credit in the fix, if you would like it.

There is no paid bug bounty at this time.
